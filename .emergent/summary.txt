    - The admin can modify subscriptions by adding/removing lessons or extending/reducing the expiry date.

- **PWA Icon:** The app should have a custom logo icon when added to a phone's home screen.
- **Booking Logic:** Past lessons should not be bookable. A lesson should remain bookable until 1 hour after its start time.
- **Coach Names:** Display the name of the instructor for each class.
- **Admin Dashboard:** A summary page for the admin showing daily statistics: total attendance, number of lessons deducted from packages, and attendance breakdown per class.
- **Auto-Cleanup:** Expired subscriptions should be automatically deleted.
- **UI/UX:**

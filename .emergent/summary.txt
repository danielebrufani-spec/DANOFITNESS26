<original_problem_statement>
The user wants to develop a comprehensive booking application for their fitness business, DanoFitness23.

**PRODUCT REQUIREMENTS:**
- **Dual Interfaces:** An admin interface for the owner (Daniele) and a client interface for customers.
- **User Registration:** Clients must register using their email and phone number. They should also be able to upload a profile picture.
- **Booking System:**
    - Bookings are made on a weekly basis, from Monday to Saturday.
    - A week can only be booked one at a time.
    - The following week becomes available for booking from the preceding Saturday morning.
- **Subscription Management (Admin only):**
    - The admin is solely responsible for creating and managing client subscriptions.
    - The admin can modify subscriptions by adding/removing lessons or extending/reducing the expiry date.
- **Internal Chat:**
    - A chat feature for communications.
    - Only the admin can initiate conversations.
    - Clients can only reply to messages from the admin.
- **Push Notifications:**
    - For new chat messages.
    - For subscription expiry:
        - 3 days before expiry for time-based subscriptions (monthly, quarterly).
        - When 2 lessons remain for usage-based subscriptions (e.g., 8 or 16 lessons pack).
- **Lesson Schedule & Activities:** The user provided a detailed winter schedule with activities (Circuito, Workout Funzionale, Pilates, Yoga), timings, and prices for various packages.
- **PWA Icon:** The app should have a custom logo icon when added to a phone's home screen.
- **Booking Logic:** Past lessons should not be bookable. A lesson should remain bookable until 1 hour after its start time.
- **Coach Names:** Display the name of the instructor for each class.
- **Admin Dashboard:** A summary page for the admin showing daily statistics: total attendance, number of lessons deducted from packages, and attendance breakdown per class.
- **Auto-Cleanup:** Expired subscriptions should be automatically deleted.
- **UI/UX:**
    - Add a confirmation dialog before any deletion action.
    - For expired users, the home screen should show an alert to contact the admin for renewal.
    - Font sizes need to be adjusted to fit well on mobile screens (Expo Go).
</original_problem_statement>
